Common Solutions

Step 1: Force Token Refresh

Firebase auth tokens are cached. Force a refresh to pick up your new permissions:

// Run this in browser console on the admin page firebase.auth().currentUser.getIdToken(true).then(() => { console.log('Token refreshed! Reload the page.'); location.reload(); });

Step 2: Clear Browser Cache

Clear all cached data and cookies:

  • Chrome/Edge: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
  • Select "Cached images and files" and "Cookies and other site data"
  • Choose "All time" for time range
  • Click "Clear data"

Step 3: Sign Out and Sign In

This ensures you get a fresh authentication session:

  1. Go to the admin dashboard
  2. Click your profile dropdown ‚Üí Logout
  3. Sign in again with claude@humanlevel.ai
  4. Check if Trees and other sections load
