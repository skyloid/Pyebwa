# Pyebwa Mobile

React Native mobile app for browsing family data, stories, and profiles.

## Stack

- Expo
- React Native
- React Navigation
- React Native Paper
- Local auth/profile persistence for development

## Scripts

- `npm start`
- `npm run android`
- `npm run ios`
- `npm run web`

## Notes

- Authentication is stored locally for this app package.
- The production web stack uses Supabase/Postgres.
- This mobile package no longer depends on the retired legacy mobile backend.
