Simple Admin Setup Tool

⚠️ Security Notice: This setup tool should only be used once to create your first admin user, then removed or restricted in production.

Setup Instructions

This tool helps you promote existing users to admin roles. Since we can't create new users directly, follow these steps:

  1. First, create a regular user account by going to the main app at /app/
  2. Come back here and enter that user's email below
  3. Select the admin role you want to assign
  4. The tool will update the user document with admin permissions

Firebase Console Method (Alternative)

If this tool doesn't work, you can manually promote users through the Firebase Console:

1. Go to Firebase Console → Firestore Database 2. Navigate to users collection 3. Find your user document 4. Add these fields: - role: "superadmin" (or "admin", "moderator") - isAdmin: true 5. Save the document
